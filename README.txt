MIRAI SPACE ウェブサイト 公開用ファイル一式
=====================================================

■ 公開（アップロード）方法
このフォルダ（dist）の中身を「すべて」、サーバーの公開ディレクトリ
（例：public_html / www / htdocs など）の直下にアップロードしてください。
index.html が自動的にトップページとして表示されます。

  例）https://www.miraixspace.jp/            → index.html（トップ）
      https://www.miraixspace.jp/service.html → サービス一覧

■ ファイル構成
  index.html ................ トップページ
  service.html .............. サービス一覧
  service-consulting.html ... コンサルティング
  service-sales.html ........ 営業支援
  service-engineering.html .. エンジニアリング
  products.html ............. 取り扱い商品一覧
  product-taitock.html ...... タイトック
  product-koone.html ........ KooNe
  product-laine.html ........ Laine
  product-capturium.html .... CAPTURIUM
  product-tl001.html ........ TL-001
  company.html .............. 企業情報
  contact.html .............. お問い合わせ
  assets/ ................... CSS・JavaScript・画像
  *.pdf ..................... KooNe 関連の資料

■ ご注意
・フォルダ構成（assets/ の中身を含む）はそのまま維持してアップロードしてください。
・「お問い合わせ」ページ（contact.html）はデザイン完成状態ですが、フォームの送信処理（メール送信・DB保存等）はサーバー設置時に接続が必要です。
・「採用情報」リンクは現行サイト（miraixspace.jp）を指しています。新URLが決まり次第差し替えます。
・文字コードは UTF-8 です。
